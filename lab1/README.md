# Конвертер цветов RGB ↔ CMYK ↔ HSV

Приложение для конвертации цветов между тремя основными цветовыми моделями: RGB, CMYK и HSV.

## Особенности

### Основной функционал
- **Конвертация между RGB, CMYK и HSV**: Точные алгоритмы конвертации между всеми тремя цветовыми моделями
- **Автоматический пересчет**: При изменении любой компоненты цвета все остальные представления автоматически обновляются
- **Предварительный просмотр**: Визуальное отображение выбранного цвета
- **Валидация входных данных**: Предупреждения о некорректных значениях с автоматическим обрезанием

### Три способа задания цвета

1. **Точные значения** - поля ввода для каждой компоненты цвета
2. **Палитра цветов** - стандартное диалоговое окно выбора цвета
3. **Ползунки** - интуитивное управление каждой компонентой

### Цветовые модели

- **RGB** (0-255): Красный, Зеленый, Синий
- **CMYK** (0-100%): Циан, Пурпурный, Желтый, Черный
- **HSV**: Оттенок (0-360°), Насыщенность (0-100%), Яркость (0-100%)

## Использование

### Выбор цвета из палитры
Нажмите кнопку "Выбрать цвет из палитры" для открытия стандартного диалога выбора цвета.

### Ввод точных значений
Введите числовые значения в поля ввода любой из трех цветовых моделей:
- **RGB**: значения от 0 до 255
- **CMYK**: значения от 0 до 100 (в процентах)
- **HSV**: H от 0 до 360 (градусы), S и V от 0 до 100 (проценты)

### Использование ползунков
Перемещайте ползунки для интуитивного изменения компонент цвета.

### Предупреждения
Приложение автоматически проверяет введенные значения и показывает предупреждения при:
- Значениях меньше минимального (автоматически обрезается до минимума)
- Значениях больше максимального (автоматически обрезается до максимума)
- Некорректном формате чисел

## Архитектура

### Классы

#### `ColorConverter`
Статический класс с методами конвертации между цветовыми моделями:
- `rgb_to_hsv(r, g, b)` - конвертация RGB в HSV
- `hsv_to_rgb(h, s, v)` - конвертация HSV в RGB
- `rgb_to_cmyk(r, g, b)` - конвертация RGB в CMYK
- `cmyk_to_rgb(c, m, y, k)` - конвертация CMYK в RGB

#### `ColorConverterApp`
Основной класс приложения, управляющий пользовательским интерфейсом и логикой обновления.

### Алгоритмы конвертации

#### RGB → HSV
```
V = max(R, G, B)
S = (V - min(R, G, B)) / V  (если V ≠ 0)
H = вычисляется в зависимости от того, какая компонента максимальная
```

#### RGB → CMYK
```
K = 1 - max(R, G, B)
C = (1 - R - K) / (1 - K)
M = (1 - G - K) / (1 - K)  
Y = (1 - B - K) / (1 - K)
```

## Дизайн интерфейса

**Верхняя половина:**
- **Круглый цветовой индикатор** (150px) с адаптивной границей
- **Кнопка палитры** для быстрого выбора цвета

**Нижняя половина:**
- **Три равные колонки** для RGB, CMYK и HSV
- **Унифицированные ползунки** (220px) с разной чувствительностью:
  - RGB: шаг 1 (0-255)
  - CMYK: шаг 0.1% (0-100%)
  - HSV: H шаг 1° (0-360°), S/V шаг 0.1% (0-100%)
- **Цветовые подсказки** в полях ввода
- **Мгновенная синхронизация** всех значений

### Особенности интерфейса
- **Адаптивные размеры окна** (900x800px по умолчанию)
- **Визуальная группировка** компонентов в рамках
- **Контрастные границы** круглого индикатора (светлые/темные в зависимости от яркости цвета)
